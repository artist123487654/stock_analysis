## 技术面分析(technology)
- 分析K线形态，各种技术指标，输出具备可操作性的股票信息和指标输出
### K线形态
- 包含各种反转形态、持续形态和特殊形态

### 技术指标
- 主要包含：
```text
    - MACD：平滑异同移动平均线指标
        金叉、死叉、零轴和背离 分析
    - KDJ ：随机动量指标
        金叉、死叉分析
    - RSI ：相对强弱指标
        金叉、死叉分析
```
